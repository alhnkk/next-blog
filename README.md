# Blog Projesi

Next.js ile geliştirilmiş modern blog uygulaması.

## Özellikler

- Next.js 14
- TypeScript
- Tailwind CSS
- Admin paneli
- Responsive tasarım

## Kurulum

```bash
npm install
npm run dev
```

## Geliştirme

Proje `http://localhost:3000` adresinde çalışır.

Admin paneli: `http://localhost:3000/admin`
